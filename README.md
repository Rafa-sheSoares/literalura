# Literalura - Backend

Projeto backend para gerenciar busca de livros usando a API Gutendex 

## Funcionalidades

- Busca de livros por autor, título ou outros filtros
- Consumo da API Gutendex para obter dados atualizados do acervo de livros
- Preparado para fornecer dados para um frontend em JavaFX (em desenvolvimento)

## Tecnologias

- Java 17
- Spring Boot
- Maven
- Lombok
- API REST externa Gutendex - https://gutendex.com/

## Como usar

1. Clone o projeto
2. Execute `mvn clean install` para compilar
3. Rode a aplicação com `mvn spring-boot:run`
4. Use os endpoints disponíveis para consultar livros via API Gutendex

## Observação

O frontend em JavaFX ainda está em processo de desenvolvimento.
